# Lynx

'Lynx' is a lightweight blog/site management engine built with [Laravel5](http://laravel.com/).

In Development Process. Release,soon.

## Features

 * Index / Archives
 * Entries
 * Posts / Pages
 * Markdown files
 * Themes
 * Plugins
 * Cache

## Requirements

 * PHP5.6 or more
 * [Composer](https://github.com/composer/composer)
