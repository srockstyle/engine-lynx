## About

'Engine Lynx' is a lightweight blog/site management engine built with [Laravel5](http://laravel.com/).

In Development Process. Release,soon.

## Features

 * Posts / Pages
 * Markdown Edit
 * Themes
 * Plugins

## Requirements

 * PHP5.6 or more
 * [Composer](https://github.com/composer/composer)
